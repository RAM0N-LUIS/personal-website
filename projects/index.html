Genomics Annotation Service

github.com/ramon-luis/genomics-annotation-service

A multi-component web application built with Python, Flask, and AWS that allows users to run genomic annotation analysis. A load balancer manages front end web server ec2 nodes that scale-in/scale-out based on web traffic triggers. The back end is comprised of ec2 nodes running the annotation jobs that scale-in/scale-out additional ec2 nodes based on the number of pending job requests. Seperate utility ec2 nodes handle archiving data, thawing/restoring data, and sending email notifications to users. Message queues and topics are used to manage system communication.

Relish news (website)

github.com/ramon-luis/relish-news

A news reader built with Ruby on Rails that allows users to save favorite newsfeeds. Users can view news headlines and articles based on topic and source, create an account to favorite topics that feed a custom home page, and toggle between viewing headlines only or article preview cards. Additionally, the application allows users to view the article text directly within the application without opening the original article.

Super Bomberman

github.com/ramon-luis/super-bomberman

A multi-threaded Java game that mimics the original Super Bomberman video game for the Nintendo Entertainment System. The player competes against the computer by attempting to use bombs to destroy enemies and clear the game map. A secret door to complete each level is revealed after all enemies have been defeated. Power-ups with special abilities are available to help the player defeat increasingly difficult enemies.

Big Data: Chicago Crime & Weather

github.com/ramon-luis/chicago-crime-and-weather

A big data web application that analyzes the relationship between weather and violent crime in Chicago daily from 2001-2016. The application uses a lambda architecture to process large amounts of data: data from multiple sources is serialized and grouped together into a single, fact-based distrubuted data table with common queries pre-computed in batch processing while a separate speed layer handles new data that has not yet been batch processed.


The application was hosted Google Cloud and built with Apache tools, including: Hadoop, HBase, Kafka, Pig, Spark, Storm, and Thrift.
