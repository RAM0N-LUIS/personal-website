Super Bomberman

github.com/ramon-luis/super-bomberman

A multi-threaded Java game that mimics the original Super Bomberman video game for the Nintendo Entertainment System. The player competes against the computer by attempting to use bombs to destroy enemies and clear the game map. A secret door to complete each level is revealed after all enemies have been defeated. Power-ups with special abilities are available to help the player defeat increasingly difficult enemies.

Jumble Jam

github.com/ramon-luis/jumble-jam

Big Data: Chicago Crime & Weather

github.com/ramon-luis/chicago-crime-and-weather

A big data project that uses lambda architecture to explore the relationship between weather and crime in Chicago. Raw weather data from the NOAA is ingested as a sequence file into a Hadoop Distributed File System (HDFS) using Java, while crime data from the Chicago public database is stored in the same HDFS in CSV format. Pig scripts pre-compute crime reports with final relevant weather and crime data joined into a single table that is grouped and stored in HBase, a non-relational, distributed database. A web application uses a Perl script to either pull existing crime & weather data or submit new crime & weather data to a Kafka message queue for processing into the Hbase table.

Picture This

github.com/ramon-luis/picture-this

An image editor built with JavaFX that allows users to edit photos. The program uses lambda expressions to take advantage of the functional programming capabilities of Java8. Implemented user features include:

  • tuning (brightness, contrast, hue, saturation)
  • grayscale
  • sepia
  • invert (colors)
  • monochromatic (colors)
  • glow effect
  • paint
  • pen
  • color picker
  • undo (20x)
  • redo (20x)
